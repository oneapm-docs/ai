1. 监控API

1.1 参数说明

  • spanTime: 最近时间
  • interval: 时间间隔
  • type: 可取值列表如下
    • WebTransaction
    • External
    • Apdex
    • Datastore
    • Database
  • start: 整数 (翻页起始位置)
  • limit: 整数 (每页显示item数目)
  • orderBy: 可取值列表如下
    • ORDERBY_S1_S2_DESC: 吞吐量
    • ORDERBY_S2_S1_DESC: 平均响应时间
    • ORDERBY_S3_S1_DESC: 平均执行时间
    • ORDERBY_S1_DESC: 调用次数
    • ORDERBY_S2_DESC: 响应时间
    • ORDERBY_S3_DESC: 调用次数
    • ORDERBY_S4_ASC: 最小值
  • endTime: 截至时间
  • fields: 可取值列表如下
    • Apdex
    • Errors
  • name: 可取值列表如下
    • WebTransaction
    • DatabaseWeb
    • MongoDBWeb
    • MemcachedWeb
    • RedisWeb
    • ExternalWeb
    • CPU
    • Mem
