Node.js agent 更新

Node.js Agent 更新方法

OneAPM Node.js Agent 更新，无需卸载 Agent 探针，再重新部署。您只需遵循以下步骤，来完成 Agent 的更新：

更新方法一

1.定位路径至你的应用程序根目录；

2.运行以下指令：

npm install oneapm

3.重启应用。

更新方法二

  1. 请前往 Node.js 安装页面，下载最新版本的 Node.js Agent。
  2. 定位路径至你的应用程序根目录。
  3. 在你的应用程序根目录下，运行以下指令：
npm install <LocationOfYourOneAPMagent> /oneapm-x.x.x.tgz

4.重启应用。
