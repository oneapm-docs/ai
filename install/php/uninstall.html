php agent 卸载

PHP Agent卸载

1、找到您的 php.ini 文件，将其保存至安全的路径。

2、进入 OneAPM 安装脚本所在目录，执行命令进行卸载

命令: ./oneapm-install uninstall

3、选择 PHP 的安装目录，如果您的 PHP 的安装目录没有在 OneAPM 探针搜索到的目录下(/usr/… 或 /opt/… 开头的目录下)，请选择 2 并用安装脚本的提示命令搜索您的安装目录

4、输入安装目录

5、回车

6、卸载完毕
