.NET agent 安装

.NET Agent 安装方法

IIS 安装步骤：

  • 登录 www.oneapm.com 官网，免费注册账号并登陆。

  • 添加应用，选择 .NET，并记录 license key 值。

  • 在 IIS 安装栏目下，选择合适的安装包。

  • 在您应用的服务器端安装Agent，安装过程中，需要输入之前记录的 license key 。

  • 重启 IIS。

  • 访问您的应用，静候几分钟，将可以在官网上开始应用性能之旅。

Non-IIS 安装步骤：

  • 登录 www.oneapm.com 官网，免费注册账号并登陆。

  • 添加应用，选择 .NET，并记录 license key 值。

  • 在 Non-IIS 安装栏目下，选择合适的安装包。

  • 在您应用的服务器端安装Agent，安装过程中，需要输入之前记录的 license key 。
